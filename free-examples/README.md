# Free Context MCP - Examples Réels

Ce dossier contient des exemples d'utilisation du MCP Free Context dans des scénarios réels.

## Structure

Chaque exemple est un scénario complet montrant comment utiliser Free Context pour gérer des connaissances et des contextes.

## Exemples Disponibles

### 1. Developer Workflow (`developer-workflow/`)
Gestion des connaissances pour un développeur : notes techniques, snippets, documentation de projet.

### 2. Research Notes (`research-notes/`)
Organisation de notes de recherche avec citations et références croisées.

### 3. Learning Journal (`learning-journal/`)
Journal d'apprentissage avec suivi des concepts et relations.

## Utilisation

Chaque exemple contient :
- `scenario.md` - Description du scénario
- `demo.sh` - Script de démonstration
- `expected-output.md` - Résultat attendu

## Tester un Exemple

```bash
cd free-examples/developer-workflow
./demo.sh
```
